Integrations

Connect your tools to make the hub fully functional. Follow each setup guide in order.

🤖
ManyChat
Instagram DM automation and keyword flows. Handles free playbook delivery, podcast drops, and speaking enquiry routing.
Setup: Create account at manychat.com → Connect @ceciliadxb Instagram → Build flows → Settings → API → Copy key → Paste below

Setup Required
🐒
Mailchimp
EDM platform for newsletters, welcome sequences, and automated nurture campaigns. Free up to 500 contacts.
Setup: Create account at mailchimp.com → Create Audience "AWAKE Subscribers" → Account → Extras → API Keys → Create → paste below

Setup Required
📱
WhatsApp Business API
Required to send WhatsApp broadcasts programmatically at scale. Connected via Meta Business Suite.
Setup: business.facebook.com → WhatsApp → Get started → Verify number → Apply for API access (2–5 business days) → Paste token below

Setup Required
⚡
ActiveCampaign
Advanced EDM alternative with better tagging, segmentation, and CRM. Use instead of or alongside Mailchimp.
Setup: Create account at activecampaign.com → Settings → Developer → API Access → Copy URL + Key → paste below

Optional Alt.
💳
Stripe Payments
Handles playbook and consultation payments. Auto-tags purchasers in the subscriber list on payment confirmation.
Setup: dashboard.stripe.com → Developers → API Keys → Restrict key → Paste below. Configure webhook to fire on payment success.

Setup Required
🔗
Zapier
Connects all your tools. Key Zaps: Website form → Mailchimp + ManyChat. Stripe payment → subscriber tag → VIP sequence.
Key Zaps to build:
1. New website form → Add to Mailchimp + tag service
2. Stripe payment → Tag subscriber as buyer
3. New ManyChat subscriber → Add to this hub

Setup Required
🗺️
How Everything Connects
The AWAKE tech stack — follow this order
Subscriber enters
Via website form, ManyChat Instagram DM, podcast link, or event QR → All routes add them here with the correct service tag
Instant delivery
Mailchimp sends the free playbook PDF automatically (Day 0). ManyChat sends a warm DM if they came via Instagram.
Nurture sequence
Mailchimp sends Day 1, 3, 7, 14 emails automatically. ManyChat sends Day 3 and Day 7 DM follow-ups.
Ongoing engagement
Regular EDM campaigns, WhatsApp community broadcasts from this hub, and podcast episode drops keep the community engaged.
Service enquiries
Keynote, masterclass, training, or collab enquiries are tagged automatically → appear in By Service view → trigger follow-up sequence
